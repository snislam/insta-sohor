How does JavaScript work?

JavaScript is an interpreted, high level, client side language. It's works in browser using a machine (in case of chrome it's V8 engine.)

It works in four steps. 1) First js code is parse to check the data error and quality and then send it to second stage. 2) In second stage is ast (Abstract syntax tree) and send to machine. 3) Machine convert the js code to machine language and 4) Proggrame execute.

No compiler is needed!
